Validating Forms

  • Form validation is a core component in web development.
  • In order to protect user's information and database integrity, we as developers, should implement client-side validation and server-side validation.
  • Theare are two types of client-side validation:
    1. Built-in form validation
    2. Javascript validation
  • Built-in form validation is performed throught HTML5 form controls and attributes
  • Another useful tool for client-side form validation is regular expression. A regular expression is a pattern that can be used to match character combinations in text strings.
